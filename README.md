# npm-repo

Redirect to an npm package's repository

`https://npm-repo.now.sh/<PACKAGE-NAME>`

## Why?

I got tired of going to `https://www.npmjs.com/package/<PACKAGE-NAME>`, browsing and clicking the repository URL.
